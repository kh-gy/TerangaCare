# 🏥 Plateforme de Téléconsultation Médicale

Projet académique — PPP DIC1 / DGI / ESP / UCAD 2025-2026

## 📋 Description
Plateforme web de téléconsultation médicale adaptée au contexte 
sénégalais, intégrant prise de RDV, carnet de santé numérique 
et orientation post-consultation.

## 🛠️ Stack technique
- **Frontend** : React.js + Vite
- **Backend** : FastAPI + SQLAlchemy + Alembic
- **Authentification** : JWT local côté backend, avec un mode développement sans auth via `APP_ENV=development`
- **BDD** : SQL relationnel
- **Vidéo** : PeerJS + WebRTC
- **Infra** : Docker + Apache 2
